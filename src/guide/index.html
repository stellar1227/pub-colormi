Button Style Guide

1. 기본 버튼

기본 버튼 스타일입니다. 배경색과 기본 여백이 적용되어 있습니다.

<button class="btn btn-default">기본 버튼</button>

2. 주요 버튼

주요 작업에 사용되는 스타일의 버튼입니다. 파란색 배경과 흰색 텍스트가 적용됩니다. 

<button class="btn btn-primary">주요 버튼</button>

3. 경고 버튼

중요한 경고를 표시할 때 사용되는 버튼입니다. 빨간색 배경으로 강조됩니다. 

<button class="btn btn-danger">경고 버튼</button>

4. 비활성화된 버튼

사용할 수 없는 상태를 나타내는 회색 버튼입니다.

<button class="btn btn-disabled" disabled>비활성화 버튼</button>
