Button 샘플

  • 너비
    • - 기본너비: 디자인가이드상 좌우여백만 부여되고 컨텐츠에 따라 늘어남.
    • - fullWidth: 부여시 부모 상속 100%
    • - width: 단위까지 추가하여 기입하면 css width에 그대로 주입
  • size
    • - medium: 기본. 사이즈 생략. font-size 14, line-height 22.4
    • - xlarge: font-size 17
    • - large: font-size 14, line-height 22.4
    • - small: font-size: 12
